✅ 1. Incorrect JSX Return Syntax

You're using double braces { { ... } }, which is invalid syntax. JSX should return a single parent element from the component, and loops like .map() should return JSX directly inside that element.

✅ 2. Incorrect placement of heading and description

You’re repeating the heading and description inside the .map() loop — so they render for every image. Instead, they should be rendered once, outside the loop.

✅ 3. Missing key on outer .map() return

You're returning a <div> inside .map() but the key={index} is on a nested element. The outermost element being returned by .map() needs the key.

All three entries have the same title/content. If it's going to be dynamic in the future, consider using an array of objects instead.
